# the_weather_flutter

A new Flutter project.

## Getting Started

Setup project with Makefile:

```
make setup
```

or manually run following commands:

```
cp .env.example .env
dart run build_runner build
```

Update your weather api key in `.env`

```
WEATHER_API_KEY=<YOUR_API_KEY>
```


Running app in emulator:
```
flutter run
```
